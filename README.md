# Codeforces Practice 

This repository contains the 16 day CP CF problems shared in the group

## Terminal commands needed {Git Bash}
### Method 1 : Without seperate Input & Output Files
-For Compiling the program
```
g++ -std=c++17 solution.cpp -o Solution.exe
```

-For Running the program
```
./Solution.exe
```

### Method 2 : With seperate Input & Output Files {Text Files}
-For Compiling the program
```
g++ -std=c++17 solution.cpp -o Solution.exe
```

-For Running the program
```
./Solution.exe < input.txt > output.txt
```
